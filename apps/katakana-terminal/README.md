# かたかな秘文ターミナル

かたかな秘文ターミナル is a browser-only retro console toy.

It pretends to decode ultra-serious missions but inevitably spits out ridiculous punchlines in Japanese.

## 遊び方

1. このディレクトリの `index.html` をブラウザで開く (サーバー不要)。
2. 起動ログを眺めつつ、光るドットボタンをテンポよくクリック。
3. 復号が完了すると、司令の正体 (だいたいギャグ) が暴かれます。
4. 「次の任務へ」で別の茶番へ進行。

## 技術メモ

- 完全クライアントサイド (HTML + CSS + vanilla JS)。
- レトロCRT風UI、ドット操作インタラクション、全UI日本語表記。
- 端末の冷静度/ノイズ指数など、ふざけたメーターを搭載。
