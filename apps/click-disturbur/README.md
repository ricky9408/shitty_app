# クリック妨害官 (Click disturbur)

メモに書かれたパスワードを「右クリック禁止・ドラッグ禁止」で5回入力させるだけの妨害系メモ打鍵ゲーム。ウィンドウズ95風の安っぽい警告音とダイアログでしつこく止められる。

## 遊び方
- 机のメモに書かれたパスワードを目で読んで `手入力フォーム` に打ち込むだけ。
- 右クリックすると警告ダイアログと95風ビープが鳴り、隠しカウンターが加算される。
- メモをドラッグ／選択してコピーしようとしても無効化され、またカウンターがこっそり増える。
- 5ラウンドで徐々に難しくなり、最後に安っぽいファンファーレが鳴る。

## 実行
- ブラウザで `apps/click-disturbur/index.html` を開くだけ（サーバ不要）。
