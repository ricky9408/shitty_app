# 希望シミュレーター24

宇宙ポッドで救難信号を送り続けるだけの希望クラッシャー系ワンクリックゲーム。日数が進むほど裏切りが増え、24日目で全てが白く溶ける。アップグレードはポッド内プリンタがスクラップを加工して“購入”扱いにしてくれる。

## 遊び方
- 画面左の「希望パルス送信」を連打して Hope を集める（一定のクリックで自動的に Day が進む）。
- Hope を使ってアップグレードを購入すると自動送信が増える。ただし後半は半減・ロックなどの罠が発生。
- Day が進むほど UI がだんだん壊れていく。24 日目で強制的に終局。
- 「緊急ノイズ送信」は Hope を大きく上下させ、送信レートも少し落ちる危険な賭け。
- アップグレードを買うと中央ダッシュボードのポッド模型に部品が装着される。全部ポッド内のスクラップ製。

## 進行の雰囲気
- Day 1-3: 正常。クリックゲーっぽい。
- Day 4-7: 数値が揺れる。値札が勝手に上がる。
- Day 8-12: 購入時に Hope が半減したり、表示が歪む。
- Day 13-17: 希望が負になる。UI がロックしたり解けたり。
- Day 18-23: 勝手に巻き戻る／揺れる／裏切る。操作遅延感。
- Day 24: 全部が白くなって Hope=∞。もう何も測れない。

## 実行
- ブラウザで `apps/hope-simulator-24/index.html` を開くだけ（サーバ不要）。
- ロジックは TypeScript (`main.ts`) で書き、ブラウザ実行用に `main.js` を置いてある。
